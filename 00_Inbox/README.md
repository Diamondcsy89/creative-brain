# 00_Inbox

这里是 Creative Brain 的临时收集区。

以后临时放进来的图片、Prompt、代码片段、网页截图、参考链接、设计想法，都先进入 `00_Inbox/`。

后续再由 Creative Engineering Agent 整理归档到对应目录，例如：

- 视觉、动效、组件、Prompt、图标和模板进入 `01_Assets/`
- 品牌资料进入 `02_Brands/`
- 项目经验进入 `03_Projects/`
- 研究资料进入 `04_Research/`
- 创意实验进入 `05_Playground/`
- AI 工作流和智能体规则进入 `06_AI/`

整理归档时，每个素材都需要补充对应的 `.md` 说明文件，包含用途、关键词、风格、Prompt、适用场景和可复用方式。
