# Visual Rules

The visual direction is a Xiaomi packaging submission page / engineering review page.

Use:

- White background.
- Light gray section backgrounds only where needed.
- Thin borders and dividers.
- Black primary text.
- Gray secondary text.
- Xiaomi orange `#FF6900` for small accents, labels, markers, and section codes.
- Tight, structured spacing.
- Small radius only, usually `4px` to `8px`.

Avoid:

- Large rounded cards.
- Cream, beige, or warm lifestyle backgrounds.
- Soft large shadows.
- Glassmorphism.
- Marketing landing-page styling.
- Decorative gradients.
- Complex platform audit tables unless explicitly requested.
- Over-designed engineering grid backgrounds.

The page should feel submit-ready, calm, precise, and reviewable.
