# Creative Brain

这是我的 Creative Brain，用于长期沉淀设计参考、Prompt、组件、动效和项目经验。

它不是一个临时素材夹，而是一个可复用的个人创意知识库：

- `00_Inbox/`：临时收集区，等待整理归档
- `01_Assets/`：视觉、动效、组件、Prompt、图标和模板资产
- `02_Brands/`：Apple、OpenAI、Xiaomi、Nothing、Leica、Tesla 等品牌参考
- `03_Projects/`：项目复盘、设计决策、技术方案和交付经验
- `04_Research/`：趋势观察、竞品拆解、资料摘要和灵感索引
- `05_Playground/`：创意实验、视觉草稿和交互原型
- `06_AI/`：AI 设计方法、模型工作流、自动化方案和智能体规则

新增素材时，请优先补充对应的 `.md` 说明文件，让每一份灵感都能被再次找到、理解和复用。
